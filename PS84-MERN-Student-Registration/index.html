🎓 PS84 — MERN Student Registration

Full CRUD Student Management System built on the MERN stack.

Step 1 — Start the Backend

Navigate to server/ folder and run:

cd server
npm install
node server.js
# → API runs on http://localhost:5000
Step 2 — Start the React Frontend

Navigate to client/ folder and run:

cd client
npm install
npm start
# → React app runs on http://localhost:3000
Step 3 — Open in Browser

Visit: http://localhost:3000

Features:
• Register students with Name, Email, Roll No, Branch, Year, Phone
• View all students in sortable table
• Edit any student (inline form pre-fill)
• Delete students (with confirmation)
• Live search by name / roll no / branch
MongoDB Express React Node.js CRUD Mongoose
