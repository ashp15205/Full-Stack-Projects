# Creative Portfolio

Developed a modern Personal Portfolio Webpage with smooth navigation, featured project showcases, and a functional contact intake system.

## Features
- **Clean Sectional Design**: Logical flow from intro hero to work history and contact.
- **Smooth Navigation**: Seamless internal linking via CSS scroll-behavior.
- **Project Grid**: Highly legible cards for displaying past technical contributions.
- **Interactive Form**: Clean contact section with validated inputs.
- **Minimalist Aesthetic**: Professional black-and-white theme focusing on project content.

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript

## Folder Structure
```text
PS23-Personal-Portfolio-Webpage/
├── index.html    # Core structure and sections
├── style.css     # Minimalist portfolio and interaction styles
└── script.js    # Smooth scroll logic and form feedback
```

## Setup & Installation
1. Open `index.html` in your browser.

## How it Works
1. The portfolio uses semantic HTML5 tags for a better SEO and accessibility footprint.
2. CSS Grid and Flexbox are used for the responsive layout.
3. The script handles small UI enhancements like navbar background logic on scroll or form success messages.

## Customization
- Replace the project information in `index.html` with your own work.
- Modify the hero text and pre-titles in the hero section to match your role.
