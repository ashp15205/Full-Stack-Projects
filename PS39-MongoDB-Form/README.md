# PS39 - Student Registration (MongoDB)
**Problem:** Register student (Name, Roll No, Branch) → store/display via MongoDB.  
**Stack:** Node.js, Express, MongoDB, Mongoose  
**Run:** `npm install && node server.js` → open `index.html`

| Method | Route | Action |
|--------|-------|--------|
| GET | `/students` | List all |
| POST | `/students` | Register new |
| PUT | `/students/:id` | Update |
| DELETE | `/students/:id` | Remove |
