# Professional Resume

Developed a Structured Resume Webpage with a professional layout, utilizing clean typography and semantic sections for optimal readability.

## Features
- **Clean Structure**: Organized sections for contact, objective, education, and skills.
- **Responsive Layout**: Designed to look professional on both desktop and mobile devices.
- **Print Optimization**: Dedicated CSS rules to ensure the resume looks perfect when printed or exported as PDF.
- **Minimalist Aesthetic**: High-contrast, black-and-white design focusing on content.

## Tech Stack
- **Frontend**: HTML5, CSS3

## Folder Structure
```text
PS19-Structured-Resume-Webpage/
├── index.html    # Resume content and layout
└── style.css     # Minimalist and print styles
```

## Setup & Installation
1. Open `index.html` in your browser.

## How it Works
1. The HTML uses semantic elements like `<header>`, `<section>`, and `<aside>` for accessible content structure.
2. The CSS uses Flexbox and Media Queries for responsiveness.
3. A `@media print` block removes the navigation and actions, ensuring only the professional content is visible on a printed page.

## Customization
- Update your personal information directly in `index.html`.
- Modify the typography in `style.css` by changing the font-family or font sizes.
